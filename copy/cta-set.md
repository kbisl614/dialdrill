# CTA Set

## Routes

- `/landing-a` — Variant A (Sales Reps, SDRs, Teams)
- `/landing-b` — Variant B (Solo Sellers, Founders, Freelancers)
- `/` — Variant C (Original copy)

---

## Primary CTAs

**Variant A (Sales Reps/Teams):**
> Start Your $5 Trial

**Variant B (Solo Sellers/Founders):**
> Try 5 Calls for $5

**Variant C (Original):**
> Start My Trial Today!

---

## Secondary CTAs

**All Variants:**
> See How It Works

---

## Alternative Primary CTAs (For Testing)

- Start Practicing for $5
- Get 5 Calls for $5
- Begin Your Trial
- Start Your First Call

---

## Notes

- Primary CTA includes price to set clear expectations and reduce friction
- "Trial" language for Variant A aligns with enterprise/team-oriented framing
- "Try" language for Variant B feels more casual and founder-friendly
- Variant C uses original copy with exclamation for energy
- Secondary CTA is identical across variants — simple, low-commitment scroll prompt
- All CTAs avoid hype language, urgency tactics, or pressure
